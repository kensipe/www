Concepts

Concepts

Framework

High-level description of a deployable application. Contains metadata about the application (e.g., Apache Kafka).

FrameworkVersion

Specific version of a deployable application, including configuration and lifecycle hooks for deployments, upgrades, and rollbacks (e.g., Kafka version 2.4.0).

Instance

Resource created to manage an instance of specific FrameworkVersion. Instances are pets and have the same name throughout its entire lifecycle. (e.g., Kafka 2.4.0 cluster with 1 broker).
